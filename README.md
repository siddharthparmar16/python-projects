# Python Beginner Practice Programs

This repository contains a collection of beginner-friendly Python programs created while learning Python fundamentals. These programs cover basic concepts such as variables, operators, conditional statements, loops, functions, mathematical calculations, and simple interactive projects.

## 📂 Programs Included

### 1. Celsius to Fahrenheit Converter
**File:** `celsius_fahrenheit.py`

Converts temperature from Celsius to Fahrenheit using the standard conversion formula.

### 2. Factorial Calculator
**File:** `factorial.py`

Calculates the factorial of a given number using Python logic and loops.

### 3. Greatest of Three Numbers
**File:** `greatest_of_3_num.py`

Finds the largest number among three user-provided numbers.

### 4. Number Guessing Game
**File:** `guessing_game.py`

A simple interactive game where the user attempts to guess the correct number.

### 5. Leap Year Checker
**File:** `leap year.py`

Checks whether a given year is a leap year according to calendar rules.

### 6. Basic Python Practice - Lines 1
**File:** `pattern_1.py`

Contains introductory Python practice exercises.

### 7. Basic Python Practice - Lines 2
**File:** `pattern_2.py`

Additional beginner Python exercises and examples.

### 8. Multiplication Table Generator
**File:** `multiplication.py`

Generates the multiplication table for a number entered by the user.

### 9. Quadratic Equation Solver
**File:** `quadratic.py`

Solves quadratic equations and finds their roots.

### 10. Sum of N Numbers
**File:** `sum_of_n_num.py`

Calculates the sum of the first N natural numbers.

### 11. Ticket Booking System
**File:** `ticket-book.py`

A simple ticket booking simulation project demonstrating basic Python programming concepts.

## 🚀 How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd <repository-name>
```

3. Run any program:

```bash
python filename.py
```

Example:

```bash
python factorial.py
```
